	\section{GPU}
		A rendelkezésemre álló NVIDIA GeForce 525M elméleti teljesítményét a következõ összefüggés alapján kaphatjuk meg:
		
		\begin{align}
			P_T = \frac{n \cdot f \cdot 2}{1000}
		\end{align}
		
		Ahol $n$ jelöli a CUDA magok számát, $f$ az órajelük frekvenciáját. A $2$ szorzó az órajel ciklusonként végrehajtott 2 darab utasítás miatt jön be.
		A legtöbb test amivel a szimulációt lefuttattam az 32768 test volt, aminél 49 GFLOPS számítási teljesítényt sikerült elérnem. A elméleti korlátnak ez sajnos csak közel a fele.
		% Elméleti határ 110 GFLOPS
		Theorethical peak performance [GFLOPS] = num of Core * Core clk[MHz] * 2 / 1000
			2, mert 2 utasítást hajt végre órajel ciklusonként
			1000 a nagyságrend miatt.
		% Profiler eredmények bemutatása
